### **Проект 5: Анализ рынка общественного питания Москвы**
**Цель:** Определить оптимальный тип и локацию для нового заведения.

**Задачи:**
- Проанализировать 8402 заведения по типу, ценовой категории и локации.
- Выявить популярные сетевые форматы и их распределение по округам.
- Оценить связь ценовой категории и рейтинга.

**Методы и инструменты:** Python (Pandas, Matplotlib, Seaborn), пространственный анализ, сегментация.

**Результат:**  
Рекомендован формат сетевого кафе/ресторана средней ценовой категории для округов с низкой долей сетей. Определены ключевые сети-конкуренты и их сильные стороны.
