### **Проект 9: Анализ сервиса доставки еды для роста лояльности**
**Цель:** Повысить лояльность пользователей и оптимизировать партнёрские программы.

**Задачи:**
- Построить воронку событий и выявить точки оттока.
- Провести RFM-сегментацию пользователей.
- Кластеризовать рестораны по бизнес-моделям.

**Методы и инструменты:** Python (Pandas, Scikit-learn), воронка, RFM-анализ, кластеризация (K-Means).

**Результат:**  
Выявлено падение мобильной конверсии на 35% и низкая лояльность (90,3% пользователей с одним заказом). Разработаны приоритетные решения по редизайну UX и программам лояльности.
