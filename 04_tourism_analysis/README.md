### **Проект 4: Исследование туристической индустрии России**
**Цель:** Проанализировать динамику туризма в России (2018–2023) для разработки персонализированных маршрутов.

**Задачи:**
- Объединить и предобработать данные о гостиницах, турпакетах и стоимости.
- Определить топ-10 регионов по стоимости турпакетов.
- Проверить гипотезы о росте внутреннего туризма и стоимости турпакетов.

**Методы и инструменты:** Python (Pandas, Matplotlib, SciPy), работа с мультитабличными данными, статистическое тестирование.

**Результат:**  
Выявлено доминирование Москвы в продажах (388,2 млрд руб.), рост внутреннего туризма после пандемии. Результаты использованы для адаптации маркетинговых стратегий сервиса.
