### **Проект 6: Анализ данных медицинского центра**
**Цель:** Оптимизировать процессы госпитализации через анализ длительности лечения и нагрузки.

**Задачи:**
- Провести предобработку данных (120 тыс. записей).
- Изучить распределение длительности госпитализации по отделениям.
- Проанализировать динамику поступлений и проверить гипотезы о длительности лечения.

**Методы и инструменты:** Python (Pandas, Matplotlib, SciPy), анализ временных рядов, статистические тесты (t-тест).

**Результат:**  
Выявлена средняя длительность лечения 8,7 дней, сезонные пики нагрузки, статистически значимая разница между плановыми и экстренными пациентами. Даны рекомендации по планированию койко-мест.
