### **Проект 12: Data-driven оптимизация маркетплейса**
**Цель:** Перевести управление маркетплейсом с интуитивного на data-driven подход.

**Задачи:**
- Провести аудит продукта, юнит-экономики и каналов привлечения.
- Проанализировать A/B-тест со стимулами для пользователей TikTok.
- Разработать стратегические рекомендации по перераспределению бюджета.

**Методы и инструменты:** Python (Pandas, Matplotlib, SciPy), A/B-тестирование, юнит-экономика, воронка.

**Результат:**  
Выявлено, что основной канал привлечения (TikTok) имеет ROI -90%, а стимулы дают статистический, но не бизнес-значимый рост. Инициирован стратегический сдвиг фокуса с конверсии на вовлечение и перераспределение бюджета.
