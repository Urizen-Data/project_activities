### **Проект 10: A/B-тест обучающего онбординга для инвестиционного приложения**
**Цель:** Оценить влияние образовательного онбординга на поведение инвесторов.

**Задачи:**
- Проанализировать исторические данные о поведении пользователей.
- Сравнить ключевые метрики (конверсия в депозит, ARPU, ARPPU) между группами.
- Исследовать распределение сумм депозитов методом бутстрапа.

**Методы и инструменты:** Python (Pandas, SciPy, Bootstrap), A/B-тестирование, анализ перцентилей.

**Результат:**  
Подтверждён рост конверсии во второй депозит на 12% и увеличение среднего чека опытных инвесторов. Онбординг рекомендован к масштабированию как инструмент сегментации аудитории.
