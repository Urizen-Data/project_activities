### **Проект 8: Исследование стартапов для выхода на рынок инвестиций**
**Цель:** Выявить признаки успешных стартапов для стратегии инвестирования.

**Задачи:**
- Объединить разрозненные данные о компаниях, раундах финансирования и сделках.
- Проанализировать типичные суммы финансирования, цены сделок и количество раундов.
- Определить привлекательные отрасли и рискованные статусы компаний.

**Методы и инструменты:** Python (Pandas, Matplotlib), работа с неполными данными, когортный анализ.

**Результат:**  
Определён типичный размер финансирования (1 млн), топ-5 привлекательных отраслей (real_estate, biotech и др.) и различия в среднем числе раундов по статусам компаний. Сформулированы стратегические рекомендации.
